## Creating ADRs

Use `/create-adr TITLE` or follow [docs/procedures/adr.md](../procedures/adr.md).

## When to Create an ADR

- Public API changes (constructor signatures, new types, removed types)
- Architectural pattern changes (reconnection strategy, shutdown pattern, backpressure mechanism)
- Deliberate trade-offs with rejected alternatives worth documenting
