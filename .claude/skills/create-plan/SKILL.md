# Create Plan

Create a new implementation plan document.

## Steps

1. Read `docs/plans/README.md` to get the next plan number and naming context
2. Create `docs/plans/<descriptive-name>.md` with template:
   - Status: Proposed
   - Overview, Goals, Tasks, Implementation Order, Acceptance Criteria
3. Update `docs/plans/README.md` index with the new entry
4. Report the created file path

## Available Tools

Bash, Read, Write, Edit, Glob
